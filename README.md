# pretzel
Greenfield Project for HackReactor

#### how to start
```shell
$ npm install
$ npm start
```

#### how to test
```shell
$ npm test
```

#### how to continuely complile when file changes
```shell
$ babel . --out-dir compiled --presets=es2015,react --ignore=node_modules,compiled,server,spec --source-maps inline -w
```
